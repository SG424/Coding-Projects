# Coding-Projects
# Python Mini Projects

This repository contains four Python projects demonstrating GUI development, game logic, error handling, and a multi-phase restaurant management system.

## Projects

### 1. Magic 8-Ball (Flet)

A simple GUI application where the user asks a yes/no question and receives a random response.

**Requirements**
- Python 3.x
- flet

Install:
pip install flet

Run:
python magic_8ball.py

---

### 2. Number Guessing Game

A console game where the user has 5 attempts to guess a randomly generated number between 1 and 50.

Features:
- Hints based on proximity
- Exception handling for invalid input
- Loop-based attempt system

Run:
python guessing_game.py

---

### 3. Plasma Reactor Stabilizer

A simple computation program that balances temperature and water input, then squares the result.

Run:
python reactor.py

---

### 4. Restaurant Management System (Phases 1–3)

A full console-based restaurant simulation including:
- Reservations and table handling
- Menu ordering system
- Tax (18% ITBIS) and tip
- Dessert menu
- Payment (cash/card)
- Receipt display
- Change calculation with Dominican denominations

Run:
python restaurant.py

---
